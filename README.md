# React-practice
React practice with enzyme and mocha

## Requirements

- Node.js (recommended: 10.x)
- npm (recommended: 6.x)

## Install

    $ git clone https://github.com/alisonjonck/react-practice
    $ cd react-practice
    $ npm install

## Run tests

    $ npm test

## Run tests with debug

    $ npm run test-debug
Open chrome://inspect on your browser

## Start & watch

    $ npm start

## Build for production

    $ npm run build
